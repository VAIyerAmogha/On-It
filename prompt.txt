Implementation instructions

Tailwind config custom theme, any CSS variables file, any globals.css

Update tailwind.config.js to map every new token into the Tailwind

Do not change any layout, spacing, component structure, font sizes,
border radii, shadows, animations, or any non-color property. If a
line of code has both a color class and a layout class, change only
the color class.
Apply the gradient tokens as inline styles or CSS custom property
references where Tailwind's bg-gradient-to-* utilities are
insufficient for the exact gradient defined above.
After replacing all tokens, do a search across the entire codebase for:

Any remaining hex color values hardcoded in JSX or CSS

as brand colors (gray used for neutral utility — e.g. text-gray-500
